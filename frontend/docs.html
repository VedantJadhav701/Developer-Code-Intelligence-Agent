Introduction

DevAgent is an execution-grounded autonomous coding agent designed for local repository orchestration.

Unlike traditional AI coding assistants that simply generate text, DevAgent operates with a full awareness of your runtime environment. It doesn't just suggest code; it executes, validates, and repairs the environment to ensure your patches actually work.

Why DevAgent?

Most coding agents fail because they lack grounding. They might propose a fix that uses a library you don't have installed, or code that uses the wrong Python version. DevAgent solves this by:

  • Detecting project requirements automatically.
  • Provisioning isolated virtual environments.
  • Autonomously repairing broken runtimes.
  • Validating every change against your actual test suite.

System Prerequisites

To run DevAgent locally with high performance, we recommend the following stack:

  • Ollama: Running the qwen2.5-coder:3b model.
  • Python 3.11+: Required for core agent logic.
  • Git: For surgical patching and safety rollbacks.
# Pull our recommended model
ollama pull qwen2.5-coder:3b

Quick Install

DevAgent is available as a single package on PyPI.

pip install devagent-cli
